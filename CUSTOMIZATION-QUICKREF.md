## Common Workflows

### Adding Text
1. Type in text input field
2. Adjust font, size, weight, color
3. Click "Add Text" button
4. Drag to position
5. Fine-tune with arrow keys

### Adding Images
1. Upload image via drag-drop or browse
2. Drag preview onto canvas
3. Resize by dragging corners
4. Rotate and flip as needed
5. Apply filters if desired

### Layering Elements
1. Select element to move
2. Use "Bring Forward" or "Send Backward"
3. Repeat until desired order
4. Elements stack with text usually on top

### Creating Duplicates
1. Select element
2. Press Ctrl+D or click Duplicate
3. Move to new position
4. Modify as needed
5. Repeat for multiple copies
